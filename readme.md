• React Fundamentals & Setup: Understanding what React is as an external library, how to load react.js and react-dom.js, and setting up a React container for rendering. This includes knowing the necessary software like a web browser (Google Chrome) and a code editor (VS Code).
• JSX (JavaScript XML): Learning this enhanced version of JavaScript that allows writing HTML directly within JavaScript code, its benefits for finding errors, and inserting JavaScript values.
• Babel: Understanding Babel as a JavaScript compiler that translates JSX into standard JavaScript that browsers can understand.
• React Components: Grasping the concept of components as reusable "pieces of the website" and how to create them using JavaScript functions that return JSX, adhering to naming conventions (Pascal case).
• Props (Properties): Learning how to pass data into components using props, which act as attributes, making components highly reusable.
• Fragments (<></>): Utilizing this special React feature to group multiple elements without adding unnecessary div elements to the HTML output, leading to cleaner code.
• Destructuring Props: Using a JavaScript shortcut to efficiently extract properties (props) from the props object within a component function.
• Conditional Rendering with Guard Operator (&&): Applying the JavaScript guard operator directly within JSX to conditionally display elements or content, similar to an if statement.
• App Component Structure: Understanding the best practice of creating a main App component that represents the entire application, allowing for further subdivision of the website into smaller components.
• Code Organization & Best Practices: Applying guidelines for formatting code, such as adding brackets around multi-line HTML, using indents, and putting attributes on separate lines for readability and maintainability.
